Current Project

  • Stochastic First-order Methods for Nonsmooth Nonconvex Optimization

    • We check the convergence performances of some first-order methods like stochastic gradient/mirror descent in nonsmooth nonconvex optimization.

    • We try to relax existing assumptions and some algorithms settings to see whether these algorithms can still converge.

    • We try to apply existing convergence results to more optimization problems

Past Projects

  • Asynchronous Multi-Task Learning (AMTL)

    • We tried to apply “Asynchronous Parallel Stochastic Gradient Methods” to Multi-Task Learning (MTL) problems.

    • My project aims to implement a prototype of this new learning framework. The main tools used are Java, Socket and some BLAS. The code is available on GitHub.

    • I worked on this project during the summer of 2016, in the Department of CMSE of MSU, supervised by Prof. Ming Yan and Prof. Jiayu Zhou
